A barebones Node.js app using [Express 4](http://expressjs.com/).

## Running Locally

Make sure you have [Node.js]

```sh

$ cd noon-api
$ npm install
$ npm start
```

Your app should now be running on [localhost:3001](http://localhost:3001/).
